# 00. Введение и настройка окружения

> Прежде чем погружаться в теорию Big Data, нужно подготовить рабочее место.
> Этот модуль — про инструменты. Без них дальше двигаться нельзя.

---

## 🎯 Цели модуля

К концу этого модуля у вас будет:

1. Установленный Python 3.10+ и менеджер пакетов `pip`.
2. Установленная Java (JDK 11 или 17) — обязательно для Apache Spark.
3. Виртуальное окружение Python с PySpark и зависимостями.
4. Запущенный Jupyter Notebook.
5. Проверенный «Hello, Spark!» — первый запуск PySpark.

---

## 📚 Содержание

1. [УСТАНОВКА.md](./УСТАНОВКА.md) — пошаговая инструкция для macOS / Linux / Windows.
2. [проверка_окружения.py](./проверка_окружения.py) — скрипт-самопроверка.
3. [hello_spark.py](./hello_spark.py) — первый запуск PySpark.

---

## ⏱️ Сколько времени займёт

~2–4 часа. Большая часть — это установка Java и Spark.
Если что-то не получается — это нормально, настройка окружения «съедает» нервы у всех, кто только начинает.
