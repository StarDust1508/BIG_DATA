# 06. Spark MLlib — машинное обучение на больших данных

> Здесь сходятся ваши два мира: ML и Big Data. Плюс правовая часть: AI Act, объяснимость, документирование.

---

## 🎯 Цели модуля

После модуля вы:

- Понимаете архитектуру `pyspark.ml`: **Transformer**, **Estimator**, **Pipeline**, **Model**.
- Строите Pipeline'ы: feature engineering → масштабирование → модель.
- Тренируете и оцениваете модели: **логистическая регрессия**, **случайный лес**, **GBT**, **KMeans**.
- Делаете **cross-validation** через `CrossValidator` и `ParamGridBuilder`.
- Сохраняете и загружаете модели, версионируете их.
- Понимаете, когда MLlib, а когда лучше **scikit-learn + pandas-UDF**.
- Документируете модель для AI Act / суда: Model Card, audit log, disparate impact.
