# DATA CARD — Legal Documents Corpus

## 1. Описание
Корпус коротких юридических документов для обучения классификатора.

## 2. Источник
- В учебном проекте: синтетика, сгенерированная `pipeline.py` из шаблонов.
- В реальном проекте: входящая корреспонденция юр.отдела за 12 месяцев, размеченная вручную.

## 3. Схема

| Колонка   | Тип    | Описание |
|-----------|--------|----------|
| text      | string | Полный текст документа |
| category  | string | Тип: иск, претензия, запрос документов, уведомление, иное |

## 4. ПДн
- Содержат ПДн: **возможно**, в реальных документах есть ФИО, ИНН, СНИЛС, адреса.
- Обработка ПДн: до обучения — обезличивание через NER (например, spaCy + правила).
- Категория ПДн: общие + потенциально специальные (национальность, здоровье — в исках).

## 5. Качество
- Сбалансированность классов: 150 документов каждого класса.
- Длина: 50-200 символов.
- Шум: переменные шаблоны.

## 6. Доступ и хранение
- Защищённое хранилище.
- ACL: только legal-tech и DPO.
- Audit log на каждое скачивание.

## 7. Жизненный цикл
- Хранение: до достижения цели обработки (обучение модели), потом архив 3 года, потом удаление.
- Удаление по запросу субъекта данных (GDPR Art. 17 / 152-ФЗ).
