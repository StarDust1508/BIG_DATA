# 🎯 Проект 2 — Классификация юридических документов

> NLP-проект для юр.отдела. Соединяет PySpark + текст + объяснимость + работа с правом.

---

## 📋 Бизнес-постановка

**Заказчик:** юридический отдел компании.

**Цель:** автоматически классифицировать входящие документы по типам:
- иск
- претензия
- запрос документов
- уведомление
- иное

Это помогает маршрутизировать в команду, оценивать срочность, готовить ответы.

**Особенности:**
- Документы — текст на русском.
- В производстве должна быть **объяснимость каждого решения** (юристам нужно знать, почему).
- На «иное» допускается ручная маршрутизация.

---

## 🏗 Архитектура

```
Входящие документы (text + метаданные)
      │
      ▼
01_load.py        — чтение CSV / Excel
      │
      ▼
02_preprocess.py  — нижний регистр, очистка пунктуации, stop-words
      │
      ▼
03_features.py    — TF-IDF + length + presence flags
      │
      ▼
04_train.py       — мультиклассовая модель (Logistic Regression)
      │
      ▼
05_explain.py     — для каждого класса топ-слов
      │
      ▼
06_inference.py   — применение к свежим документам
      │
      ▼
Parquet + MODEL_CARD.md + explanations.json
```

## 🛠 Что нужно установить

Минимум: модули 00 + 02 + 04 + 06. Всё в `requirements.txt`.

Бесплатно полностью.

Для более серьёзного NLP (морфология, лемматизация) — `pymorphy3`:
```bash
pip install pymorphy3
```
В курсе используем только базу — splitting + stop-words + TF-IDF.

## ▶️ Запуск

```bash
cd 07_проекты/проект_2_юр_документы
python3 pipeline.py
```

Сгенерирует синтетический корпус и обучит модель. Результаты:
- `MODEL_CARD.md` — карточка модели с метриками и топ-словами.
- `datasets/legal_docs_predictions.parquet` — предсказания.

---

## ⚖️ Правовая часть

Документы могут содержать ПДн (ФИО участников). Обрабатываем:

- [ ] Корпус хранится в защищённой зоне.
- [ ] Перед обучением — псевдонимизация ФИО через NER (в реальном проекте).
- [ ] Категория AI Act: **ограниченный риск** (классификация — не automatic decision-making).
- [ ] Объяснимость: для каждого предсказания возвращается топ-слов с весами.
- [ ] Human override: если модель не уверена (max_prob < 0.6) → класс «иное» → ручная маршрутизация.
