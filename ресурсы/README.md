После прохождения модулей 01–07 рекомендуется читать в таком порядке:

1. **[АУДИТ_СЛАБЫХ_МЕСТ.md](../АУДИТ_СЛАБЫХ_МЕСТ.md)** — понять, чего курс не дал.

3. **[глубокий_sql.md](./глубокий_sql.md)** — без SQL не пройдёте интервью.
4. **[spark_internals.md](./spark_internals.md)** — понимание Spark глубже.
5. **[тестирование_spark.md](./тестирование_spark.md)** — без тестов production не бывает.
6. **[performance_practice.md](./performance_practice.md)** — реальные кейсы тюнинга.
7. **[streaming_kafka_spark.md](./streaming_kafka_spark.md)** — если идёте в streaming.
8. **[mlops_углублённо.md](./mlops_углублённо.md)** — если идёте в ML.
9. **[probabilistic_ds.md](./probabilistic_ds.md)** — для senior-уровня.
10. **[реальные_кейсы.md](./реальные_кейсы.md)** — для широкого видения.
11. **[интервью.md](./интервью.md)** — за неделю до собеседования.
12. **[ДАЛЬШЕ_ПОСЛЕ_КУРСА.md](../ДАЛЬШЕ_ПОСЛЕ_КУРСА.md)** — выбрать карьерный путь.
