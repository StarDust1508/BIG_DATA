## 3. Транзакция / парадигма MapReduce

```mermaid
flowchart LR
    Input[(Большой файл<br/>в HDFS/S3)]
    Input --> Split[Split<br/>на блоки]

    Split --> M1[Map task 1]
    Split --> M2[Map task 2]
    Split --> Mn[Map task N]

    M1 --> KV1[k1, v1]
    M2 --> KV2[k2, v2]
    Mn --> KVn[kN, vN]

    KV1 & KV2 & KVn --> Shuffle{Shuffle &amp; Sort<br/>по ключу}

    Shuffle --> R1[Reduce task 1]
    Shuffle --> R2[Reduce task 2]
    Shuffle --> Rm[Reduce task M]

    R1 --> Out[(Результат<br/>в HDFS/S3)]
    R2 --> Out
    Rm --> Out

    style Shuffle fill:#ffe1e1
    style Out fill:#e1ffe1
```

## 4. Lazy evaluation в Spark

```mermaid
sequenceDiagram
    participant U as Пользователь
    participant DF as DataFrame
    participant C as Catalyst
    participant E as Executors

    U->>DF: spark.read.parquet("...")
    Note over DF: ничего НЕ читается
    U->>DF: .filter("amount > 100")
    Note over DF: только план
    U->>DF: .groupBy("city")
    Note over DF: только план
    U->>DF: .agg(...)
    Note over DF: только план

    U->>DF: .show()
    Note over DF: ACTION!
    DF->>C: построй оптимальный план
    C->>C: predicate pushdown<br/>column pruning<br/>broadcast hint
    C->>E: распредели tasks
    E->>E: читает Parquet,<br/>фильтрует, агрегирует
    E-->>U: результат
```

## 5. ETL pipeline (идеомпотентный)

```mermaid
flowchart TD
    Source[(Источник:<br/>CSV / DB / API)] --> Extract[Extract<br/>validate schema]
    Extract -->|fail schema| Alert1[🚨 Schema alert]
    Extract -->|ok| Clean[Clean<br/>dedup, types, nulls]

    Clean --> DQ1[DQ check<br/>not empty<br/>no negative<br/>no orphan]
    DQ1 -->|fail| Alert2[🚨 DQ alert]
    DQ1 -->|ok| Pseudonymize[Псевдонимизация<br/>SHA-256 + salt]

    Pseudonymize --> Enrich[Enrich<br/>broadcast join clients]
    Enrich --> Aggregate[Aggregate<br/>by segment/category]

    Aggregate --> DQ2[Final DQ]
    DQ2 -->|ok| Write[(Write Parquet<br/>partitionBy=dt<br/>dynamic overwrite)]
    DQ2 -->|fail| Alert3[🚨 Hold]

    Write --> Audit[Audit log<br/>kpi, rows, hash]
    Write --> Downstream[Downstream:<br/>BI / ML / API]

    style Source fill:#e1f5ff
    style Write fill:#e1ffe1
    style Alert1 fill:#ffe1e1
    style Alert2 fill:#ffe1e1
    style Alert3 fill:#ffe1e1
```
