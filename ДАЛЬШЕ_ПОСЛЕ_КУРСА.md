## 1. Где вы сейчас (после курса)

Вы можете:
- Уверенно работать с PySpark на средне-крупных данных.
- Строить ETL-пайплайны и базовые ML-модели.
- Читать чужой Spark-код, понимать архитектуру.

Это уровень **junior data engineer** или **ML engineer** с фокусом на Big Data. Для middle нужны 1–2 года реального проекта + ниже-перечисленные специализации.

## 2. Три карьерных трека

Big Data — широкая профессия. Выберите вектор:

### 🛠 Data Engineer

Фокус: ETL, пайплайны, хранилища, инфраструктура.

**Что учить:**
- **Airflow / dbt** — оркестрация и трансформации.
- **Kafka + Spark Streaming** — реальный realtime.
- **Lakehouse**: Delta / Iceberg глубже.
- **Cloud DWH**: Snowflake, BigQuery, Redshift.
- **Docker + Kubernetes** для деплоя.
- **CI/CD** для data: GitHub Actions, GitLab CI.

**Зарплата (РФ, 2025–2026):**
- Junior: 120–180k.
- Middle: 200–350k.
- Senior: 400k+.

### 🤖 ML Engineer / MLOps

Фокус: production-ML, инференс, мониторинг моделей.

**Что учить:**
- **MLflow** — управление экспериментами и моделями.
- **Feature Store** (Feast, Tecton).
- **Online inference**: TensorFlow Serving, KServe, Ray Serve.
- **Monitoring**: data drift, model drift (Evidently, NannyML).
- **PyTorch / TensorFlow** для нейросетей.
- **A/B-тестирование** ML-моделей.

**Зарплата:**
- Junior: 130–200k.
- Middle: 220–400k.
- Senior: 450k+.

### 📊 Analytics Engineer

Фокус: данные «как продукт» для бизнеса. На стыке DE и DA.

**Что учить:**
- **dbt** — главный инструмент.
- **Глубокий SQL** + оконные функции + рекурсивные CTE.
- **Метрики Layer**: Cube, Lightdash, Metabase.
- **BI**: Tableau / Power BI / Metabase / Superset.
- **A/B-тестирование** и продуктовая аналитика.

**Зарплата:**
- Junior: 100–160k.
- Middle: 180–280k.
- Senior: 350k+.

## 3. Roadmap на год после курса

### Месяц 1–2: Закрепить Spark
- Сделать **публичный pet-project** на основе модуля 07.
- Выложить на GitHub с хорошим README.
- Решить 20–30 задач на StrataScratch / DataLemur по SQL.

### Месяц 3–4: Airflow + dbt
- Книга «Data Pipelines with Apache Airflow».
- Туториал dbt на их сайте — бесплатный.
- Перепишите проект 3 на Airflow + dbt.

### Месяц 5–6: Облако
- Зарегистрируйтесь в Databricks Community (бесплатно).
- Пройдите 1–2 их курса.
- AWS / GCP / Yandex Cloud — освойте основы.
- Если хотите серьёзно — **AWS Cloud Practitioner** (~ $100).

### Месяц 7–8: Углубление по треку
- DE: Streaming с Kafka и Flink.
- ML: PyTorch + transformer'ы.
- AE: dbt advanced + продуктовая аналитика.

### Месяц 9–10: Сертификации
- **Databricks Spark Developer Associate** — стоит времени, $200.
- **Yandex Cloud Engineer** — бесплатно (Yandex Practicum часто проводит).
- **AWS Data Analytics Specialty** — $300, серьёзный.

### Месяц 11–12: Job hunt
- Резюме с упором на проекты, а не курсы.
- Mock interviews (Pramp, Interviewing.io).
- LeetCode SQL + algorithms (для middle/senior).

## 4. Что почитать (приоритет)

```
- Стек: PySpark, Spark MLlib (Random Forest), Parquet, Delta Lake
- Особенность: автоматическая генерация Model Card, обезличивание ПДн через
  SHA-256+salt, audit log
- Метрики: ROC-AUC 0.81 на синтетическом датасете
- GitHub: <ссылка>

Проект 2: NLP-классификация юр.документов (5 классов)
- Стек: PySpark + TF-IDF + Logistic Regression
- Объяснимость: топ-слов на каждый класс
- F1 0.85 на 5-классовой задаче
- GitHub: <ссылка>

Проект 3: ETL-пайплайн для регулярной отчётности
- Стек: PySpark с idempotent partition overwrite, schema validation, DQ-проверки
- Оркестрация: примеры в Airflow / Prefect / Dagster
- GitHub: <ссылка>
```

Это работает в 100 раз лучше, чем «прошёл Coursera курс».

## 7. Юридический трек (отдельная ветка для юриста)

Раз вы юрист — у вас есть возможность редкая для рынка:

### Legal Tech / Compliance Tech

Стэк, который вас сделает уникальным:
- Big Data + ML + юр.экспертиза.
- Работа: банки, страховые, КИИ, госсектор.

**Что добавить:**
- **152-ФЗ и GDPR на уровне эксперта** (не просто читать — практический опыт DPO).
- **AI Act EU** — пока единицы в РФ понимают глубоко.
- **Кибербез-сертификации** (ISACA CISM, CIPP/E).
- **NLP для документов** — pymorphy, BERT для русского, NER.

**Где работать:**
- Юр.отделы крупных IT/банк/страховых (Sber, Tinkoff, Yandex).
- Compliance в международных компаниях.
- LegalTech-стартапы.
- Консалтинг (KPMG, Deloitte, EY имеют legal tech практики).

**Зарплата:**
- Compliance / Privacy officer с пониманием ML: 250–500k.
- Legal Engineer / LegalTech Lead: 300–700k.
