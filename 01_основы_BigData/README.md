# 01. Основы Big Data

> Прежде чем писать код, разберёмся в концепциях. Если перескочить этот модуль, дальше будет ощущение «делаю что-то, не понимаю что».

---

## 🎯 Цели модуля

После этого модуля вы:

- Понимаете, что такое «большие данные» и чем они отличаются от «обычных».
- Знаете модель **5V** и умеете применять её к реальным датасетам.
- Ориентируетесь в **экосистеме Big Data**: Hadoop, Spark, Kafka, NoSQL, Data Lake/Warehouse/Lakehouse.
- Понимаете идею **распределённых вычислений**, теорему **CAP** и почему она важна.
- Различаете **форматы хранения** (CSV, JSON, Parquet, ORC, Avro) и знаете, когда какой брать.
- Видите **правовой контекст** Big Data: 152-ФЗ, GDPR, AI Act.

## 🛠 Что нужно установить

Модуль преимущественно **теоретический** + один Python-скрипт.

Достаточно того, что вы поставили в модуле 00:
- Python 3.10+ с активным venv
- Пакеты из `requirements.txt` (pandas, pyarrow — для практики 2)

Проверить: `python 00_введение/проверка_окружения.py` — должно быть зелёное.

Если нет — вернитесь к [УСТАНОВКА.md](../00_введение/УСТАНОВКА.md). Бесплатные альтернативы (Colab, Kaggle) — в [среда_разработки.md §4](../ресурсы/среда_разработки.md#4-облака).
