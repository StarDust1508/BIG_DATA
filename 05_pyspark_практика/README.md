# 05. PySpark — практика

> Дальше — много кода. Берём реальные паттерны: ETL, join'ы, окна, партиционирование, обезличивание, тюнинг. Это самый большой и самый «прикладной» модуль курса.

---

## 🎯 Цели модуля

После модуля вы:

- Пишете полноценные **ETL-пайплайны** в PySpark: чтение → очистка → трансформации → запись Parquet с партициями.
- Уверенно используете **join'ы** (включая broadcast).
- Применяете **window functions** для running-агрегатов, ranking, lag/lead, sessionization.
- Видите **skew** и умеете его лечить (AQE / salt-trick).
- Используете **caching** там, где это даёт выигрыш.
- Реализуете **псевдонимизацию ПДн** в pipeline через встроенные `F.sha2`.
- Знакомитесь с **Delta Lake** (DELETE/UPDATE/MERGE/Time Travel).

## 🛠 Что нужно установить (бесплатно)

Базовое — то же, что в модуле 04: Python + Java + PySpark.

### Что добавится в этом модуле

**Delta Lake** (урок 5.9, практика с Lakehouse):

```bash
pip install delta-spark==3.2.0
```

В коде:
```python
from delta import configure_spark_with_delta_pip
builder = SparkSession.builder.appName("...").config("spark.sql.extensions",
    "io.delta.sql.DeltaSparkSessionExtension")
spark = configure_spark_with_delta_pip(builder).getOrCreate()
```

`delta-spark` подтянет нужные JAR'ы. Бесплатно.

### Опциональное

- **dbt-spark** (`pip install dbt-spark`) — если хотите попробовать индустриальный фреймворк трансформаций. В курсе не используется, но в реальных пайплайнах — стандарт.
- **Airflow** (`pip install apache-airflow`) — оркестрация ETL. Сложный, ставить только если планируете изучать оркестрацию.

### Профессиональный совет

В реальных проектах вы будете запускать Spark-pipeline'ы через `spark-submit`, а не `python script.py`. Это полезно попробовать:

```bash
spark-submit --master local[*] --driver-memory 2g практика_1_etl.py --run-date 2026-05-15
```

`spark-submit` идёт вместе с PySpark — отдельно ничего ставить не надо.

### Облачно

- **Databricks Community** — самое то для модуля 05. Delta Lake встроен.
- **Google Colab** — тоже подходит, но без персистентности Parquet (нужно сохранять в Google Drive).
